## 一、文档概述

### 1.1 产品背景

为幼儿园教师/教务人员提供一套可视化的周排课工具。系统左侧为可复用的活动资源库，右侧为周一至周五的课表网格。用户可在编辑模式下拖拽活动到具体时段，也可预览已排好的计划。

### 1.2 产品目标

- 降低幼儿园周计划排课的操作成本。
- 让课程、负责人、日主题、时段说明在同一界面内完成编排。
- 提供快速调整工具（如整日对调），应对临时调课需求。

### 1.3 目标用户

| 角色 | 核心诉求 |
|------|----------|
| 班主任/任课教师 | 快速拖拽生成周计划、查看活动详情、指定负责人 |
| 教务管理员 | 预览全园/全年级计划、复制计划、批量调整 |

## 三、全局状态：编辑态 vs 预览态

### 3.1 状态定义

| 状态 | 入口 | 左侧活动面板 | 课表交互 | 右上角按钮 |
|------|------|--------------|----------|------------|
| **预览态** | 页面默认状态 | 隐藏 | 只读，可查看详情 | 编辑、复制计划 |
| **编辑态** | 点击「编辑」进入 | 显示，可拖拽 | 可拖拽排课、编辑日计划、整日对调 | 保存、取消、Ai帮排 |

### 3.2 状态切换规则

- 进入编辑态：点击右上角「编辑」按钮，左侧活动面板展开，课表进入可编辑模式。
- 退出编辑态：
  - 点击「保存」：保存当前修改并回到预览态，弹出提示「计划已保存」。
  - 点击「取消」：放弃本次编辑并回到预览态，弹出提示「已取消编辑」。
- 当前演示版未实现真实的撤销/重做数据栈，「取消」仅用于切换状态。

## 四、功能需求

### 4.1 顶部导航

#### 4.1.1 周切换

- **上一周**：课表日期整体前移 7 天。
- **回到本周**：日期跳转至系统当前日期所在周。
- **下一周**：课表日期整体后移 7 天。

#### 4.1.2 日期直达输入框

- 输入格式：`yyyy-MM-dd`。
- 输入后自动跳转到该日期所在周的周一。
- 输入非法日期时，输入框边框变红并提示「请输入正确日期格式」。

#### 4.1.3 右上角操作按钮

- **编辑**（预览态）：点击进入编辑态。
- **复制计划**（预览态）：占位按钮，点击提示「复制计划功能开发中」。未来用于复制当前周计划到下一周或生成副本。
- **保存**（编辑态）：保存并退出编辑态。
- **取消**（编辑态）：不保存并退出编辑态。
- **Ai帮排**（编辑态）：占位按钮，点击提示「Ai帮排功能开发中」。未来用于基于规则/历史数据智能推荐排课方案。

---

### 4.2 左侧活动面板

#### 4.2.1 活动列表

- 每个活动展示：名称、领域标签、颜色背景。
- 悬浮时显示详情按钮 ⓘ，点击打开活动详情弹窗。
- 编辑态下卡片可拖拽；预览态下不可拖拽。

#### 4.2.2 活动筛选

- **来源筛选**：全部 / 我的 / 园所 / 平台。
- **领域筛选**：全部 / 健康 / 语言 / 社会 / 科学 / 艺术。
- **类型筛选**：全部 / 生活 / 运动 / 游戏 / 学习。
- 多组筛选为**交集**关系。
- 搜索框：按活动名称模糊匹配，支持一键清空。

#### 4.2.3 空态

- 当筛选/搜索无结果时，显示「未找到相关活动」。

---

### 4.3 右侧课表

#### 4.3.1 日期表头

- 显示日期（MM-DD）与星期（周一 ~ 周五）。
- 当天日期高亮显示（蓝色下划线、蓝色日期文字）。
- 若设置了自定义今日主题，在表头下方显示蓝色主题标签。
- 鼠标悬浮表头时，出现 Tooltip 展示「本月主题 / 本周主题 / 今日主题」。

#### 4.3.2 课程单元格

- 每个单元格对应一个日期 + 一个时段。
- 空单元格：可接收拖拽活动。
- 已排课单元格：显示活动卡片、负责人下拉框、时段说明（如有）。
- 编辑态下：
  - 单元格可接收拖拽。
  - 拖拽到空单元格：直接安排该活动。
  - 拖拽到已有活动单元格：弹出替换确认弹窗，确认后替换活动但**保留原负责人**。
  - 点击单元格（非卡片区域）：打开每日编辑弹窗。
- 预览态下：只读，不可拖拽。

#### 4.3.3 负责人设置

- 每个已排课单元格提供「负责人」下拉选择。
- 候选负责人：王老师、李老师、张老师、陈老师、赵老师、刘老师、周老师、吴老师。
- 切换负责人后弹出 Toast：「已设置负责人：XXX」。

#### 4.3.4 活动详情查看

- 点击活动卡片或课表内活动卡片上的 ⓘ 按钮，打开活动详情弹窗。
- 详情包括：活动领域、活动类型、活动来源、活动简介、教学目标、课程资料链接。

---

### 4.4 每日编辑弹窗

- 入口：编辑态下点击日期表头或课程单元格（非卡片区域）。
- 可编辑内容：
  - **今日主题**：文本输入框。
  - **各时段说明**：每个时段一个文本输入框（如晨间说明、备注）。
- 操作：保存 / 取消。
- 保存后，表头主题标签和单元格内的时段说明即时更新。

---

### 4.5 替换确认弹窗

- 触发：编辑态下拖拽活动到已有活动的单元格。
- 弹窗文案：「该时段已有活动 [旧活动]，是否替换为 [新活动]？」
- 操作：
  - **替换**：新活动替换旧活动，原负责人保留。
  - **取消**：放弃替换，提示「已取消替换」。

---

### 4.6 整日对调（长按日期格拖拽）

#### 4.6.1 触发条件

- 仅在编辑态生效。
- 用户在日期表头格子上**长按约 500ms** 后触发拖拽。

#### 4.6.2 交互流程

1. 长按源日期格，源格缩小并半透明，鼠标旁出现拖拽幽灵提示「交换 MM-DD 星期X 排课」。
2. 拖动到目标日期格，目标格高亮。
3. 松开鼠标/手指，两天所有时段的课程、负责人、日主题、时段说明**整体对调**。
4. 弹出 Toast：「已交换 MM-DD 与 MM-DD 的排课」。

#### 4.6.3 约束

- 自然日顺序（周一到周五的列位置）保持不变，仅交换单元格内数据。
- 若拖动到非日期格或源格本身，不执行交换。
- 短按日期格仍保持原有行为：打开每日编辑弹窗。
- 支持触摸屏：`touch-action: none` 防止长按过程中页面滚动。

## 五、数据模型

### 5.1 活动（Activity）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一标识 |
| name | string | 活动名称 |
| type | string | 领域：健康 / 语言 / 社会 / 科学 / 艺术 |
| category | string | 类型：生活 / 运动 / 游戏 / 学习 |
| source | string | 来源：my / institution / platform |
| duration | number | 时长（分钟） |
| color | string | UI 主题色 |
| teacher | string | 默认教师 |
| description | string | 活动简介 |
| targetAge | string | 适用年龄段 |
| objectives | string | 教学目标 |
| materials | array | 课程资料列表（含名称与 URL） |

### 5.2 排课数据（ScheduleData）

- 键：`date|timeSlot`，例如 `2026-06-16|09:00-10:00`。
- 值：
  - `activity`：活动对象。
  - `assignedPerson`：负责人姓名。

### 5.3 日计划数据（DailyEditData）

- 键：`dateStr`，例如 `2026-06-16`。
- 值：
  - `theme`：今日主题。
  - `slots`：对象，键为时段，值为时段说明。

### 5.4 主题数据（Themes）

- 按月/周维护，包含：
  - 月份、月主题。
  - 周编号、周开始/结束日期、周主题。
  - 每日主题映射表。
- 自定义日主题优先级高于系统主题。

## 六、非功能需求

### 6.1 性能

- 当前为前端静态演示，数据量小，无分页需求。
- 后续接入真实数据后，活动列表建议支持虚拟滚动（>50 条时）。

### 6.2 兼容性

- 支持现代浏览器（Chrome、Safari、Edge、Firefox）。
- 支持桌面端鼠标与移动端触摸操作。

### 6.3 可用性

- 所有操作提供 Toast 反馈。
- 关键操作（替换、对调）需要二次确认或明确结果提示。
- 编辑态与预览态通过按钮颜色和左侧面板显隐清晰区分。

### 6.4 安全

- 当前为纯前端演示，无后端接口。
- 后续接入后端时，需对活动资料 URL、日主题内容做 XSS 过滤。

### 7.2 布局

- 顶部导航固定高度，主体区域占满剩余视口。
- 左侧活动面板宽度 320px，右侧课表自适应。
- 预览态下左侧隐藏，课表区域适当加宽。

### 7.3 反馈

- Toast：顶部居中，2 秒后自动消失。
- Modal：居中弹窗，点击遮罩可关闭。
- Tooltip：表头悬浮时展示主题信息。

## 九、后续可扩展方向

1. **Ai帮排**：基于教师偏好、活动类型不冲突、场地资源等约束自动推荐排课方案。
2. **复制计划**：支持复制本周到下周、复制单日到多日、生成计划模板。
3. **冲突检测**：同教师同一时段重复安排时自动标红提醒。
4. **打印/导出**：支持导出为 PDF 或图片，方便张贴。
5. **历史版本**：保存排课历史，支持回滚。
6. **多人协作**：教师之间共享计划、评论与审批。
